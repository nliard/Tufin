Instructions

  • Domains, Zones, USPs and Exceptions are loaded on startup if online. Click the reload button if you want to refresh.
  • Enter Domain where you want to create USPs.
  • You can start from the suggested clean USP with 3 zones, select an existing Zone from Tufin to edit or import a local USP.
  • To import a local USP, just paste it's contents in the textarea below and click on Import CSV.
  • Click on the Zone names on the left to rename.
  • Click on cells to modify policy.
  • After finising your edits, click on Export CSV to generate the USP contents in the textbox.
    • To load the results into Tufin you need to copy and paste all generated contents into a .CSV text file and import into a new or existing USP on SecureTrack->Audit->Compliance.
  • To import a local USP, just paste it's contents on below textarea and then click on Import CSV button.
Online functions require server installation because of Same-Origin

Exceptions

  • Cells will be marked if an exception exists.
  • Click on a cell to view or to add a new exception
  • Exceptions are added directly to the server via API

About USP Generator and Editor


Available in hosted form via CodePen: http://codepen.io/packetinspector/full/PbYpXL/.

You can edit this in codepen too, just choose the editor view at http://codepen.io/packetinspector/pen/PbYpXL. Note: Does not currently support the number attributes of rule properties (Days, Count).
They are all normalized to "X", please update that yourself in the textbox for now if it is in use.

Tips: You can keep the edit dialog box open and click cells to edit. You'll still need to save if you make a change though.

Get the files, the script and more information at https://github.com/packetinspector/Tufin/tree/master/USP-Generator.
